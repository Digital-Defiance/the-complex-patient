A Gutenberg block that wraps a Font Awesome Icon in a configurable link.

== Description ==

Easy Links for Font Awesome provides a block editor block that places the native Font Awesome Icon block inside a link wrapper. You get all the original icon options (picker, styling, animations, color, size) plus full link controls:

* URL with WordPress link search
* Open in new tab
* Custom rel attribute
* Link title
* Aria label for accessibility
* Additional CSS classes
* Border, typography, spacing, shadow, and color controls

This plugin is not affiliated with or endorsed by Font Awesome.

== Installation ==

1. Ensure the [Font Awesome](https://wordpress.org/plugins/font-awesome/) plugin is installed and activated.
2. Upload the `easy-links-fa` folder to `/wp-content/plugins/`.
3. Activate the plugin through the Plugins menu in WordPress.
4. In the block editor, search for "Easy Links for Font Awesome" to insert the block.

== Frequently Asked Questions ==

= How do I access all the settings? =

Use the List View or breadcrumb bar to select the outer "Easy Links for Font Awesome" block. The inner Font Awesome Icon block handles icon selection and styling. The outer block handles link settings, border, typography, spacing, and additional CSS classes.

= Does this require Font Awesome Pro? =

No. It works with both the free and pro versions of the Font Awesome plugin.
